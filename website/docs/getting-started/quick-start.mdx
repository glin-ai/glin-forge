# Quick Start Guide

Deploy your first smart contract on GLIN Network in 5 minutes. This guide will walk you through creating, building, and deploying an ERC20 token contract.

## Step 1: Create a New Contract

Create a new token project using the ERC20 template:

```bash
glin-forge new my-token --template erc20
```

### Expected Output

```
Creating new contract project: my-token
âœ“ Template: erc20
âœ“ Created directory: my-token
âœ“ Generated Cargo.toml
âœ“ Generated lib.rs

Next steps:
  cd my-token
  glin-forge build
```

### What Just Happened?

GLIN Forge created a new directory with:
- `Cargo.toml` - Project configuration
- `lib.rs` - ERC20 contract source code
- `.gitignore` - Git ignore rules

Navigate to your project:

```bash
cd my-token
```

## Step 2: Build the Contract

Compile your contract to WebAssembly:

```bash
glin-forge build --release
```

The `--release` flag optimizes the build for smaller size and lower gas costs.

### Expected Output


### What's Inside?

Let's peek at the generated files:

<Tabs>

</TabItem>

## Step 3: Deploy to Testnet

Deploy your contract to GLIN testnet:

```bash
glin-forge deploy \
  --network testnet \
  --account alice \
  --args "1000000,MyToken,MTK"
```

### Constructor Arguments

The `--args` parameter passes:
- `1000000` - Initial supply (1 million tokens)
- `MyToken` - Token name
- `MTK` - Token symbol

## Step 4: Interact with Your Contract

Now let's interact with the deployed contract.

### Query Total Supply (Read-only)

Query functions don't cost gas:

```bash
glin-forge query $CONTRACT_ADDR totalSupply
```

### Expected Output

```
Querying contract...
âœ“ Connected to testnet

Result: 1000000
```

### Check a Balance

Check Alice's token balance:

```bash
glin-forge query $CONTRACT_ADDR balanceOf 5GrwvaEF5zXb26Fz9rcQpDWS57CtERHpNehXCPcNoHGKutQY
```

### Expected Output

```
Result: 1000000
```

Alice has all 1 million tokens since she deployed the contract!

### Transfer Tokens (Transaction)

Transfer 1000 tokens to Bob:

```bash

  5FrLwHLX9Tfa2e8hXr8PUYvjCPYA5fBxQ6fTVVQRzBqN8kBy \
  1000 \
  --account alice \
  --wait
```

<TabItem value="hooks" label="React Hooks">

```typescript title="types/useMyToken.ts"
import { useContract } from '@glin-ai/sdk-react'

export function useMyToken(address: string, signer?: any) {
  const { contract, loading, error } = useContract({
    address,
    abi: metadata,
    signer,
  })

  return { contract, loading, error }
}
```

</TabItem>
<TabItem value="usage" label="React Usage">

```tsx title="App.tsx"
import { useMyToken } from './types/useMyToken'

function TokenBalance({ address }: { address: string }) {
  const { contract, loading } = useMyToken(address)
  const [balance, setBalance] = useState<bigint>(0n)

  useEffect(() => {
    if (contract) {
      contract.query.totalSupply().then(setBalance)
    }
  }, [contract])

  if (loading) return <div>Loading...</div>

  return <div>Total Supply: {balance.toString()}</div>
}
```

```rust
// Add a mint function
#[ink(message)]
pub fn mint(&mut self, to: AccountId, amount: Balance) -> Result<()> {
    let caller = self.env().caller();
    // Add admin check...
    self.total_supply += amount;
    self.balances.insert(to, amount);
    Ok(())

```

Then rebuild and redeploy.

</TabItem>
<TabItem value="test" label="Add Tests">

Run the included tests:

```bash
glin-forge test
```

Add your own tests:

```rust
#[ink::test]
fn transfer_works() {
    let mut contract = MyToken::new(1000, "Test".into(), "TST".into());
    let accounts = ink::env::test::default_accounts::<ink::env::DefaultEnvironment>();

    assert!(contract.transfer(accounts.bob, 100).is_ok());
    assert_eq!(contract.balance_of(accounts.bob), 100);
}
```
